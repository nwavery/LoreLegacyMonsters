# Third-Party License Review (Steam v1.0)

Owner: Release Producer  
Last updated: 2026-05-04

## Scope

This checklist tracks legal review and release readiness for bundled third-party software and model artifacts.

## Components

### Ollama runtime

- [ ] Exact distributed version recorded.
- [ ] MIT license text copied into release notice package.
- [ ] Source attribution URL included.
- [ ] Binary integrity/signature verified.

### Llama 3.2 model

- [ ] Exact model variant recorded (size/quantization).
- [ ] Llama Community License reviewed by legal.
- [ ] Distribution/hosting terms approved for Steam depot distribution.
- [ ] Required attribution statements included.
- [ ] Any restricted use clauses reflected in EULA/policy text.

### Steamworks SDK / Steamworks.NET

- [ ] SDK version pinned and documented.
- [ ] Redistribution rules reviewed.
- [ ] Required notice text included.

## Release gate

- [ ] `Assets/StreamingAssets/oss-notices.txt` generated by `scripts/Generate-OssNotices.ps1`.
- [ ] Final notices reviewed by legal.
- [ ] EULA and Privacy Policy updated to reflect bundled local AI runtime.
