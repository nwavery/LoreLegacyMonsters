# Launch Day Runbook

## T-24h

- [ ] Final RC build uploaded to release candidate branch.
- [ ] Verify RC install + boot on clean machine.
- [ ] Confirm store page assets and copy approved.
- [ ] Confirm support staff coverage windows.

## T-2h

- [ ] Promote release branch/depot to default branch.
- [ ] Validate storefront visibility and release time.
- [ ] Publish launch post draft to Steam news/Discord.

## Launch window (first 4 hours)

- [ ] Monitor crash logs, support inbox, and community hub.
- [ ] Triage top issues by severity.
- [ ] Confirm achievement unlock and cloud save signal.
- [ ] Post known-issues thread if needed.

## First 72 hours

- [ ] Daily hotfix decision checkpoint.
- [ ] Refund-risk monitoring with issue taxonomy.
- [ ] Update FAQ for recurring support questions.
- [ ] Post transparent patch notes for every hotfix.
