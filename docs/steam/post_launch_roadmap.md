# Post-Launch Roadmap (Draft)

## Patch cadence

- Week 1-4: weekly hotfix window.
- After month 1: bi-weekly patch cadence.
- Use Steam beta branches for staging before default branch promotion.

## Metrics

- Crash-free session percentage.
- Steam Cloud sync error rate.
- Achievement unlock completion funnel.
- Median time-to-first-LLM-response.
- Support ticket volume by category.

## v1.1 target

- Mac (Apple Silicon) support.
- Linux support and Steam Deck verification pass.
- Steam Input iteration and controller UX polish.

## v1.2 target

- Optional alternate local model choices.
- Runtime model switching UX.
- Performance pass for first-response latency.

## v1.3 target

- Localization framework rollout.
- First non-English language pack.
- Full string externalization and QA pass.
