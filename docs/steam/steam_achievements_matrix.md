# Steam Achievement Matrix (v1.0)

This matrix defines the initial Steam achievement set and corresponding runtime IDs.

## Core progression

- `ach_first_battle` - first battle completion
- `ach_first_evolution` - first evolution
- `ach_chapter_one_complete` - complete chapter one return quest
- `ach_chapter_two_complete` - complete chapter two return quest
- `ach_chapter_three_complete` - complete chapter three return quest
- `ach_phase_two_complete` - complete phase two binding choice arc

## Branch outcomes

- `ach_iona_spare` - choose Iona spare branch
- `ach_iona_withdraw` - choose Iona withdraw branch
- `ach_corin_truth` - reveal Corin truth branch line
- `ach_varo_refuse_spire` - choose Varo refuse-spire branch
- `ach_advisor_chosen` - lock advisor selection

## Endings

- `ach_ending_merge` - choose Merge ending
- `ach_ending_seal` - choose Seal ending
- `ach_ending_replace` - choose Replace ending
- `ach_ending_burn` - choose Burn ending

## Collection milestones

- `ach_capture_ten` - capture 10 monsters total
- `ach_capture_fifty` - capture 50 monsters total
