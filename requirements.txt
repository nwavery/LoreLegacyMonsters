# Lore, Legacy, and Monsters Project Requirements

## Core Requirements
- **Unity 6000.0.x** (Unity 6; match `ProjectSettings/ProjectVersion.txt`, e.g. **6000.0.41f1**)
- Saves are **JSON** on disk (`SaveSystem` / `SaveLoadManager`) — **SQLite is not used** in this codebase
- C# / .NET as configured by the Unity 6 project

## Recommended Development Tools
- Visual Studio 2022 or newer, or JetBrains Rider
- Git for version control
- **Ollama** (optional but required for alpha LLM testing) — see `README.md` and `scripts/Setup-Ollama.ps1`

## Installation (developers)
1. Clone this repository
2. Open the project in the **same Unity 6 editor version** as `ProjectVersion.txt`
3. Run Edit Mode tests from **Window → General → Test Runner**

## Alpha builds (internal)
- Menu: **Build → Alpha → Windows Standalone (64-bit)**
- Or: `.\scripts\Build-Alpha.ps1 -UnityPath "C:\Path\To\Editor\Unity.exe"`
- Tester-facing notes: `ALPHA_TESTING.md`, `SMOKE_CHECKLIST.md`
